# Caso-de-Estudio
Caso de Estudio: “Sistemas de Búsqueda para una Biblioteca Digital Estudiantil”
#### Miembros: Diego Silva y Andres Mejia

## Contexto
La Universidad desea desarrollar un prototipo inicial de un sistema de búsqueda para su futura
Biblioteca Digital Estudiantil. El sistema debe permitir localizar información de manera
eficiente dentro de una colección de datos como libros, autores, categorías y códigos.
Como programadores en formación, se les ha asignado la tarea de investigar, diseñar e
implementar algoritmos de búsqueda simples en C#, que puedan sentar las bases de este
proyecto.
## Objetivo del caso de estudio
Aplicar los conceptos de búsqueda en estructuras de datos mediante algoritmos implementados
manualmente, fomentar el trabajo colaborativo y documentar la investigación usando normas
APA.

# Instrucciones de uso
Dividimos el proyecto en ramas, en donde cada una corresponde al tipo de busqueda que se le quiera dar a la bibloteca. Para su uso solo hacer pull al la rama interesada, ejecturar y listo.
